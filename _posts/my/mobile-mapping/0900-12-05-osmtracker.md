OSMTracker
==============

> ဤ လမ်းညွှန်ချက်ကို OSMTracker_my.odt  (သို့) OSMTracker_my.pdf  တွင် download လုပ်နိုင်ပါသည်။  
> Reviewed 2016-02-08
  
ဤဆောင်းပါသည် OSMTracker for Android နှင့် ဆက်စပ်မှုရှိပါသည်။ Wiindows mobile အတွက် အလားတူ product တစ်ခု ရနိုင်ပါသည်။   
OSMTracker သည် လမ်းကြောင်းတစ်လျှောက် waypoints များ စုစည်းမှနှင့် အတူ ခရီးတစ်ခု gpx လမ်းကြောင်း ၏ ဖန်တီးမှုကို ခွင့်ပြုပေးသည်။ အသံသွင်းခြင်း၊ ဓါတ်ပုံရိုက်ခြင်းနှင့် တခြားမှတ်စုများကို recoard လုပ်ပေးပြီး geolocate လုပ်ပေးသည်။

အခုပဲ စတင်လိုက်ပါ။
-----------

![OSMTracker_1][]  

(၁) လမ်းကြောင်းအသစ် ဖန်တီးရန် ကြက်ခြေခတ်လေးကို နှိပ်ပါ။  
(၂) ခလုပ်များပြရန် screen ပြောင်းလဲသွားပါလိမ့်မည်။ GPS တပ်ဆင်မှုပြီးကြောင်းပြသော မီးခိုးရောင်ပျောက်သွားသည်အထိစောင့်ပါ။  
(၃) မြေပုံလုပ်ရန် တိကျသော feature ကိုရောက်သောအခါ သင့်တောင်သောခလုပ် ကိုဖိထားပြီး သင်၏လမ်းကြောင်းကိုသွားပါ။  
(၄) သင်ပြီးဆုံးသောအခါ သင်၏ track နှင့် waypoints များသိမ်းမှတ်ရန် floppy disk icon ကို ဖိထားပါ။  
(၅) Track ကို gpx file တစ်ခုကို ပြောင်းရန် list ထဲမှ ထိုအရာကို ဖိ၍ကိုင်ထားပါ။ ထို့နောက် သင့်တော်သော option ကိုရွေးပါ။  
(၆) Track ၊ ဓါတ်ပုံ နှင့် recording တို့ကို သင်၏ကွန်ပျူတာကိုပြောင်း ၍ OSM ကို Update လုပ်ရန် ထိုအရာတို့ မှ ဖတ်ပါ။  


OSM Tracker ကို Install လုပ်ပါ။
-------------------------

Google play store (https://play.google.com/store/apps/details?id=me.guillaumin.android.osmtracker) မှ OSM tracker ကို Install လုပ်ပါ။  
![OSMTracker Logo][]  
အကယ်၍ လိုအပ်လျှင် မကြာသေးခင်က ထွက်ရှိထားသော application package(https://drive.google.com/folderview?id=0BxxhTXmYjyeSSjg1MFhJWnJLams#list)  ကို Google play store ပြင်ပမှ Download လုပ် နိုင်ပါသည်။  


OSMTracker ဖွဲ့စည်းခြင်း
------------------------

OSM Tracker ကို စပါ။ သင် ၏ Android device မှ Menu icon ကို ဖိထားပြီး Setting ကိုရွေးချယ်ပါ။  

Voice record duration - စာရေးသူအနေနဲ့ ကတော့ ၃၀ မိနစ်အသုံးပြုသည်။ သို့သော်လဲ ဆန္ဒရှိသလောက် record လုပ်နိုင်သည်။  
Enable sound - ဤ အကွက်ကို အမှတ်ခြစ်ရန် အကြံပြုပါသည်။ Voice recording အစနှင့် အဆုံးကို အချက်ပေးရန် “beep” အသံထုတ်လွှတ်သည်။  
Reset authentication – uploaded လုပ်ပြီးသာ gpx file နှင့် OpenStreetMap အကောင့် ကို သင်၏ လမ်းကြောင်းများ Upload လုပ်ရန် ခွင့်ပြုထားသော OSMTracker ရှိသ၍ ဤခလုတ်သည် grey out ဖြစ်သွားပါလိမ့်မည်။  
GPS settings - တည်နေရာကို ဆုံးဖြတ်ရာတွင်အသုံးပြုရန် သင်၏ Android device တွင် နည်းလမ်းပြထားပါသည်။ မှန်ကန်တိကျမှုရှိရန် အကြံပြုထားပါသည်။ device သည် GPS, Wifi နှင့် Mobile networks တို့ကို တည်နေရာ ခန့်မှန်းရန် အသုံးပြုပါလိမ့်မည်။  
Check GPS at startup – OSMTracker ကိုစတင်သောအခါ GPS ပိတ်ထားပါက Programme သည်ပြန်ဖွင့်ရန်သင့်ကိုချက်ချင်းလှုံ့ဆော်ပါလိမ့်မည်။
Ignore GPS clock – GPS အချက်ပေးချက် အတွင်းမှ ထောက်ပံ့ထားသောအချိန်အစား device တွင်ရှိသော Internal clock ကိုအသုံးပြုခဲ့လျှင် အမှတ်ခြစ်လိုက်ပါ။  
GPS logging interval – recording ၏ အခြေအနေ ကြား အချိန်ကာလကိုရွေးပါ။ အကောင်းဆုံးတိကျမှုအတွက် ၀ (သို့) ၁ကိုရွေးပါ။ အဘယ့်ကြောင့်ဆိုသော်စက္ကန့်တိုင်းကို recording လုပ်ပေးသည်။  

> ကြိမ်နှုန်းသည် ဘက်တ္ထရီသုံးပြုမှုကို အကျိုးသက်ရောက်မှုရှိစေသော်လည်း ဤ application ကို အသုံးပြုသောအခါ အချိန်အများစုတွင် သင်၏စက် မျက်နှာပြင် (screen) သည်ပွင့်နေပါလိမ့်မည်။ ဥပမာ။  ။ GPS အလုပ်လုပ်ခြင်း၊ ဓါတ်ပုံရိုက်ခြင်း၊ recording လုပ်ခြင်း စသည်တို့သည် သင့် ဘက်တ္ထရီကိုပြုန်းတီးစေနိုင်ပါသည်။ ထို့ကြောင့် သင်၏ စက်ကို အားအပြည့်ဖြည့်ထားရန် အကြံပြုပါသည်။  

- အပိုသိုလှောင်မှု External storage (SD) directory  

> မကြာသေးမီကမှထွက်ရှိထားသော Android devices များသည် external microSD (သို့) SD ကဒ်ထဲတွင် data သိမ်းမှတ်မှုကို ခွင့်မပြုပါ။  “external storage” ဟု ရည်ညွှန်းထားသော်လည်း tracks များသည် /storage/emulated/0/osmtracker. ထဲတွင်သာ သိမ်းမှတ်ဆဲဖြစ်ပါလိမ့်မည်။   

- One directory per track - Voice recordings, photographs, text notes နှင့် the gpx trace တို့အားလုံးသည် gpx file ဖန်တီးမှု ၏ အချိန်နှင့်နေ့စွဲ ကိုအမည်ပေးထားသော တူညီသော folder ထဲတွင် သိမ်းမှတ်ထားရန် အကြံပြုထားပါသည်။  
- Filename for named tracks - ဖိုင်များကို အမည်ပေးခြင်းအတွက် ဆန္ဒရှိလျှင်သင့်အကြိုက်ရွေးချယ်မှတ်သားနိုင်သည်။  
- Accuracy in GPX file - မီတာအနည်းငယ်မျှ အကွာ မှားယွင်းမှုပါဝင်သော waypoint အမည်ပါဝင်တက်ပါသည်။ ဥပမာ။ ခြေလှမ်း (၄မီတာ) ဤကဲ့သို့သော သတင်းအချက်အလက်များသည် OSM ကို Update လုပ်သောအခါနှင့် trace နှင့် waypoints အသစ်များ ကို feature တစ်ခု သို့ရွှေပြောင်းရန် ချိန်ညှိရာတွင် ဆုံးဖြတ်ခြင်းရန်အလွန်အရေးပါပါသည်။   
- Use HDOP approximation - နေရာကို ပိုမို မှန်ကန်လာစေရန် ထပ်ဆောင်း တက်ချက်မှုများနှင့် ဆက်စပ်နေသည်။  
- Export compass Heading – Compass data သည် GPX file ကို မည်သို့ export လုပ်သည်ကို အဓိပ္ပာယ်ဖွင့်ဆိုထားသည်။  
- Default photo source - ယေဘုယျအားဖြင့် ဤ app မှတစ်ဆင့် ကင်မရာဖြင့် ဓါတ်ပုံရိုက်ပါလိမ့်မည်။ သို့သော် ဖုန်းထဲရှိသော ဓါတ်ပုံများသာ အသုံးပြု ၍ရနိုင်မည်ဖြစ်သည်။  
- Screen always on - ခလုတ်များအဆက်မပြတ်နှိပ်ခြင်း၊ ဓါတ်ပုံရိုက်ခြင်း (သို့) အသံသွင်းခြင်းများပြုလုပ်ပါက Screen ကို အမြဲ ဖွင့်ထားပါ။ အကယ်၍ထိုအရာများမလုပ်ပါက ဘတ္ထရီစားသက်သာစေရန် Screen ကို ဗလာလုပ်ထားပါ။  
- Main screen graphic theme – ခလုတ်များကို ကြည့်ရာတွင် အဆင်မပြေပါက contrast နှင့် Brightness ကိုပြောင်းရန် ရွေးချယ်မှုများဖြစ်သည်။  
- Button Presets - သင်၏ကိုယ်ပိုင် ခလုတ်များကို ဒီဇိုင်းဆွဲပြီး Installလုပ်နိုင်ပါသည်။ ဤ option ကို အသုံးပြုပြီး မျက်နှာပြင်ပေါ်၌ပြထားရန် ရွေးချယ်ပါ။ဤ လုပ်ဆောင်ချက်၏ ရှင်းလင်းချက်အပြည့်အစုံကို အောက်တွင်ကြည့်ပါ။   
- Background Map - ဤfeature ကို သိမ်းမှတ်ထားသော track ပေါ်တွင် ဖိထားခြင်းဖြင့် ရွေးချယ်ပြီးပါက track ကို မျက်နှာပြင်ပေါ်၌ပြရပါလိမ့်မည်။ ထို့နောက် app သည် OpenStreetMap data များကို နောက်ခံအနေဖြင့် ဒေါင်းလုပ်လုပ်ပါလိမ့်မည်။  

> မြေပုံကိုဒေါင်းလုပ်လုပ်ရန် အင်တာနက် လိုအပ်ပါသည်။ သင်ဆန္ဒမရှိပါက ဤ လုပ်ဆောင်ချက်ကို သုံးစရာမလိုပါ။  

- Map Tile Provider - မြေပုံကို မျက်နှာပြင်ပေါ်သို့ ပြထားသောအခါ map data ကို နောက်ခံအနေဖြင့် ဒေါင်းလုပ်လုပ်ရန်ရွေးချယ်ပြီးသောအခါ နောက်ခံအနေဖြင့် သင်သည် မြေပုံ၏ version ကို ရွေးရမည်။  
- Orientation - မျက်နှာပြင်ကို rotate (သို့) Lanscape နှင့် Portrait ကိုရွေးလိုပါသလား။  

လမ်းကြောင်းစာရင်း
--------------

![OSMTracker_2][]  
ခလုတ်များနှိပ်ခြင်း၊ အသံဖမ်းမှုများဖန်တီးခြင်း နှင့် ဓါတ်ပုံရိုက်ခြင်း စသည့် အစပိုင်းခြုံငုံလေ့လာမှုပြုလုပ်ပြီးပါက သင်၏ track ကို အဆုံးသတ်ရန် floppy disk icon ကို နှိပ်ပါ။ ယခုစာရင်းထဲမှ လမ်းကြောင်းကို နှိပ်ပါ။ ရွေးချယ်မှုများ ရရှိလာပါလိမ့်မည်။  

- သင်အနည်းငယ်လွတ်သွားသည်ကို သတိပြုမိပါက Tracking ကို ဆက်လက်လုပ်ဆောင်ပါ။  
- Display သည်လမ်းကြောင်းကို သင် ၏data connection ပေါ်မူတည်ပြီး background map တစ်ခုနှင့် Background map တွင် သင်ရွေးချယ်ထားသော option နှင့်တကွ ပြပါလိမ့်မည်။  
- GPX အနေနှင့် Export လုပ်ပါ။ Programme ကို တည်းဖြတ်ရာတွင် အသုံးပြုရန် track နှင့် waypoint အတွက်လိုအပ်ပါသည်။  
- OpenStreetMap ကို upload လုပ်ပါ။ သင်၏ trace ကို OpenStreetMap ပေါ်သို့ Upload လုပ်ရာ တွင်ဤ option ကိုသုံးပါ။ သင့်ထံတွင် OpenStreetMap အကောင့်တစ်ခုရှိရန်နှင့် သင်၏အကောင့်ပေါ်သို့ tracesများ upload လုပ်ရန် ဤ app ကို တရားဝင်ခွင့်ပြုချက်ရ ထားရန် လိုပါသည်။ (How to obtain explained here) OpenStreetMap အတွက် ဤ traces များ အဘယ့်ကြောင့်အရေးကြီးကြောင်း သိလိုပါက GPS Traces - gpx files below တွင်ကြည့်ပါ။  

> သတိပြုရန်။ သင်၏ track ကို Delete မလုပ်သေးရန်။ သင်သည် အခြေခံလမ်းကြောင်းသာ upload လုပ်ပြီးခြင်းဖြစ်သည်။ waypoints နှင့်  ဓါတ်ပုံများ မပါဝင်သေးပါ။ တည်းဖြတ်ခြင်းအတွက် ကွန်ပျူတာထဲသို့ ပြောင်းရွှေ့ပြီးသည်အထိအားလုံးထိန်းသိမ်းထားပါ။  

- မိမိကိုယ်တိုင်ရှင်းပြချက်တို့အားဖျက်ပစ်ပါ။  


ခလုတ်အဆင့်မြင့်တပ်ဆင်ခြင်း
--------------

သီးခြား အကြောင်းပြချက်တစ်ခုကြောင့် ခြုံငုံလေ့လာမှု (survey) လုပ်လိုပြီး သင်လိုအပ်သော waypoint labal မရရှိပါက  သင်၏ ခြုံငုံလေ့လာမှုပြုလုပ်သောခရီးမတိုင်မှီ ခလုတ်များကို ကြိုတင် preset လုပ်သွားပါ။ သင်၏ဖုန်းထဲတွင် ဖတ်သွားပြီး survey လုပ်သောအခါ ရွေးချယ်ပါ။ သင်လိုအပ်သောအခါ သင်၏ဖုန်းထဲမှ ဖတ်နိုင်ပြီး သင်၏ မှတ်တမ်းတင်မှု ကိုမရပ်နားစေပဲလှည့်လည်အသုံးပြုနိုင်သည်။ သင်ဖန်တီးလိုက်သော label သည် သင် ၏ကိုယ်ရေးကိုယ်တာမှတ်စုတစ်ခုထဲတွင်ရှိသောကြောင့် OpenStreetMap အတွင်းအသုံးပြုသော tags များနှင့် လိုက်လျောညီထွေရှိအောင်ပြင်ဆင်ရန်မလိုပါ။ Upload လုပ်ရန်လဲမလိုပါ။ ဥပမာ။ ခလုတ်ရှိ Labels များကို သင်ရွေးချယ်နိုင်သည်။ အောက်တွင်ကြည့်ပါ။

GPS လမ်းကြောင်းများ - gpx files 
----------------------  

![OSMTracker JOSM][]  

Gpx files အနေနှင့်ရှိသော GPS traces များသည် အစိတ်အပိုင်းများနှင့် ပြုလုပ်ထားသည်။ စည်းမျဉ်းစည်းကမ်းအရ trace နှင့် waypoints များဖြစ်သည်။ အပေါ်တွင်ပြထားသော screenshot သည် OSMtracker ထဲမှ folder တစ်ခုဖြစ်သည်။ trace ၏အလည်ဗဟိုမှ စတုရမ်းပုံ icon လေးသည် ဤတည်နေရာ၌ ရိုက်ထားသောဓါတ်ပုံတစ်ပုံကို ညွှန်ပြခြင်းဖြစ်သည်။ ဓါတ်ပုံကို မျက်နှာပြင်ပေါ်၌ ကြည့်လိုလျှင် စတုရမ်းပုံ icon လေးကို နှိပ်လိုက်ပါ။  

သင့်ကို လမ်းညွှန်ချက်များပေး ရန် GPS ကို သုံးပြီး လမ်းညွှန်ပြမှုအောက်တွင်သွားနေပါက ပြန်လည်တွက်ချက်ရမည့် နေရာအချို့ကို သင်သတိပြုမိလိမ့်မည်။ အဘယ့်ကြာင့်ဆိုသော်အတွင်း၌ရှိသော GPS တည်နေရာသည် သူမျှော်လင့်ထားသောလမ်း (သို့) လမ်းကြောင်းနှင့် ညီညွှတ်မှုမရှိသောကြောင့်ဖြစ်သည်။ အမှားအတွက် မာဂျင်မျဉ်းတစ်ကြောင်းရှိသည်။သို့သော် လမ်းနှင့် လမ်းကြောင်းများအတွက် မှားယွင်းသောနေရာတွင်ဖန်တီးရန်လွယ်ကူသည်။ (သို့မဟုတ်) လမ်းကိုmap data ဖန်းတီးပြီးသောအခါမှသာ ရွှေ့ပြောင်းရန်ဖြစ်သည်။   
OpenStreetMap ကို update လုပ်သောသူများသည် ဧရိယာတစ်ခုတွင် upload လုပ်ရန် trace များအားလုံးကို ဒေါင်းလုပ်လုပ်ပြီး အောက်ပါအခြေအနေများတွင်အသုံးပြုနိုင်ပါသည်။  

(၁) တိမ်များ၊ သစ်ပင်များ၊ အရိပ်များ၊ အဆောက်အအုံများစသည်တို့ကြောင့် satellite ကိုအသုံးပြု၍လမ်းကြောင်းများမြင်ရမည်မဟုတ်ပါ။  
(၂) ဂြိုလ်တုကို မှန်ကန်စွာချိန်ညှိနိုင်ရန် ဂြိုလ်တုပုံကို ထိမ်းညှိပါ။ ပုံတချို့နှင့်အတူ အံ့ဩဖွယ်ကောင်းသော ပြောင်းလဲမှုများလည်းရှိနိုင်သည်။  

Single Gpx trace တစ်ခုသည်အသုံးဝင်သော်လည်း ထိုအရာသည်သီးခြား မှန်ကန်တိကျနိုင်မည်မဟုတ်သလို မီတာ 30  သာသာလည်းကွာဟမှုရှိနိုင်သည်။ traces အများအပြားရှိပါက မှန်ကန်သောလမ်းကြောင်းကို ယခု လွယ်ကူစွာတွေ့မြင်နိုင်ပါသည်။ ဥပမာ ။ အောက်တွင်တွေ့ရသော မြို့တစ်မြို့၏ ဒေါင်းလုပ်လုပ်ထားသော gpx traces ကိုကြည့်ပါ။ လမ်းကြောင်းအထူအပါးကိုမူတည်၍ လမ်းမကြီးကို သိနိုင်ပါသည်။
